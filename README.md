# How use this code

## This code have 4 parameters:
Position: start_time and end time animation; start_pos and end_pos surface; easing your animation
Rotation: start_time and end time animation; start_anglr and end_angle surface; easing your animation
Scale: start_time and end time animation; scale for surface; easing your animation
Transparency: start_time and end time animation; start_transparency and end_transparency surface; easing your animation
